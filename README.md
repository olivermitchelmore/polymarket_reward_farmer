# Polymarket Reward Farmer

Collects liquidity rewards by placing and holding orders at a predefined spread on configured markets.

## Status

Work in progress. This project is under active development and not ready for use. Functionality is incomplete and untested.

## Features

- Real-time market data via WebSocket
- Configurable order sizes and spreads per market
- Automatic order placement and cancellation
- Multi-market support
- Exposure management
